# Intro to RNA-seq Analysis

This project walks through a beginner-friendly RNA-seq workflow, including QC, alignment concepts, and differential expression analysis.

## Steps Included
1. FASTQ quality control (FastQC)
2. Alignment overview (STAR/BWA explanation)
3. Count matrix generation
4. Differential expression with DESeq2
5. Visualization (volcano plot, heatmap)

## Goal
To understand the basic structure of an RNA-seq pipeline and practice analysis in R.
